# Offensive Security Lab

This repository documents my hands-on journey in offensive security.

## 🎯 Objective
To simulate real-world attack scenarios, understand how they work, and explore detection and remediation strategies.

## 🔐 Focus Areas
- Active Directory attacks (Kerberoasting, AS-REP Roasting, privilege escalation)
- Lateral movement techniques
- Detection and security analysis
- System security fundamentals

## ⚙️ Lab Environment
- Windows Server (Domain Controller)
- Windows Client
- Kali Linux

## 🧠 Approach
For each attack:
- Understand how it works
- Execute it in a controlled lab
- Analyze the impact
- Propose mitigation strategies

## 🚀 Goal
To build a strong foundation in offensive security and develop a practical understanding of real-world security challenges.
